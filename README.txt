
# SPK Mobil CI3

### Cara menggunakan
1. Download CodeIgniter 3 dari https://codeigniter.com/download
2. Ekstrak folder `system` dan `index.php` ke dalam folder ini
3. Atur `application/config/config.php` -> base_url
4. Atur `application/config/database.php` -> koneksi database
5. Import database `database/spk_mobil.sql`

Folder ini sudah berisi controller, model, view, dan struktur siap pakai.
